# Future Mobile Packaging Guide

The frontend is designed so Capacitor can wrap it later without rewriting the app.

Android:

- Add Capacitor.
- Configure microphone permission for voice input.
- Configure notification permission for notices.
- Test Chrome and Samsung Internet behavior before packaging.

iOS:

- Add Capacitor iOS platform on macOS.
- Add microphone and speech usage descriptions.
- Test safe-area, keyboard, back navigation, and standalone display.

Store readiness:

- Replace temporary icons with approved university-branded assets.
- Add privacy policy URL.
- Confirm official permission for UNIMATE branding and data use.
